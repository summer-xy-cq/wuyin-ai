# 更新日志 (CHANGELOG)

## [1.0.0] - 2026-02-11

### 已完成功能
- **体质评估系统**：实现了基于五音理论的中医体质自测问卷。
- **AI 音乐生成与推荐**：集成 AI 接口，根据评估结果生成或推荐个性化调理音乐。
- **可视化图表**：使用 ECharts/Chart.js 展示详细的体质分布结果。
- **个人中心**：管理个人体质记录和历史数据。
- **UI/UX 优化**：完成响应式布局设计，支持移动端和深色模式。
- **部署**：项目已成功部署至 Vercel。

### 技术栈
- Vue 3 + Vite
- Tailwind CSS
- ECharts & Chart.js
- Lucide Vue Next (图标库)
- OpenAI/Gemini SDK
